# Workshop-Kotlin

![GitHub Logo](https://media.giphy.com/media/C4NdKtRaQE9m8/giphy.gif)

Workshop - Treinamento Linguagem Kotlin

Os links abaixo correspondem aos exercícios propostos e desenvolvidos pela JetBrains. O site possui uma Web IDE com suporte a linguagem Kotlin. 

# Instruções: 

1 - Clone este repositório

2 - Crie uma branch com o seguinte template:  codelab/nome

3 - Escreva o seu código no Site online para testar a solução e salve um arquivo .kt com o nome do exercício. 

4 - Abra um pull-request para a branch master para avaliação.
